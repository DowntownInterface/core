
# Downtown

A UI/UX Library made by Parsvel built on shadcn/ui. Made for Parsvel tools and application to conform to company design and theme.

Downtown was based on Tailwind CSS and Radix UI. Made for React.

## Features

- Solid design
- Fast and efficient
- Light/Dark mode
