
# Downtown

A UI/UX Library made by Parsvel with user experience and design in mind. Downtown was made for Parsvel tools and application.

Downtown was based on Tailwind CSS and Radix UI. Made for React.

## Features

- Solid design
- Fast and efficient
- Light/Dark mode
